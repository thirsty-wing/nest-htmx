# Nest-htmx

Project primarily targeted at being a htmx project, and just so happens to be written using nestjs. A POC to see if htmx can fulfill needs.

## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
